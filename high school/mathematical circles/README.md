# Mathematical Circles: Russian Experience

This folder contains my solutions and notes for the book **"Mathematical Circles: Russian Experience"** by Dmitri Fomin, Sergey Genkin, and Ilia Itenberg.

---

## 📚 Source

**Book:** *Mathematical Circles: Russian Experience* — Fomin, Genkin, Itenberg

This book is a collection of mathematical problems and ideas from Russian mathematical circles — extracurricular groups where students explored mathematics beyond the standard curriculum.
